# One-pager: 웹 기반 평면도 디자인 툴

## 1. TL;DR
CAD 툴을 다룰 수 없는 일반 사용자들이 웹 브라우저에서 간단하게 평면도를 그리고 가구를 배치할 수 있는 직관적인 디자인 툴입니다. 방 리모델링이나 가구 재배치를 계획하는 사람들이 복잡한 소프트웨어 없이도 시각적으로 공간을 설계할 수 있도록 돕습니다.

## 2. Goals
### Business Goals
* 기존 CAD 시장에서 소외된 일반 소비자 시장 진입
* 웹 기반 SaaS 모델로 안정적인 수익 구조 구축
* 인테리어/리모델링 관련 업체와의 파트너십 기회 창출
* 사용자 생성 콘텐츠를 통한 커뮤니티 플랫폼 구축

### User Goals
* 복잡한 CAD 프로그램 없이 평면도 작성
* 가구 배치 시뮬레이션을 통한 사전 계획 수립
* 리모델링 아이디어를 시각적으로 구체화
* 가족이나 전문가와 쉽게 공유 가능한 결과물 생성

### Non-Goals
* 전문 건축가용 고급 CAD 기능 제공
* 3D 렌더링이나 고도의 시각화 기능
* 건축 법규나 구조적 계산 기능
* 모바일 앱 네이티브 개발 (웹 우선)

## 3. User stories
**주요 페르소나: 김민수 (35세, 직장인, 신혼부부)**
* 새 아파트 이사 후 가구 배치를 미리 계획하고 싶다
* 아내와 함께 공간 활용에 대해 논의하고 싶다
* 가구 구매 전 실제 공간에 맞는지 확인하고 싶다

**보조 페르소나: 이영희 (42세, 주부, 리모델링 계획 중)**
* 아이방을 서재로 바꾸는 리모델링을 계획 중이다
* 인테리어 업체와 상담 전 구체적인 계획을 세우고 싶다
* 예산에 맞는 최적의 공간 활용 방안을 찾고 싶다

## 4. Functional requirements
### 핵심 기능 (Phase 1)
* 드래그 앤 드롭 방식의 평면도 그리기
* 벽, 문, 창문 등 기본 건축 요소 추가
* 침대, 소파, 테이블 등 기본 가구 라이브러리
* 치수 표시 및 실제 크기 반영
* PNG/PDF 형태로 결과물 다운로드

### 고도화 기능 (Phase 2)
* 가구 크기 커스터마이징
* 색상 및 텍스처 옵션
* 템플릿 라이브러리 (아파트 유형별)
* 협업 기능 (링크 공유, 댓글)
* 개인 프로젝트 저장 및 관리

### 확장 기능 (Phase 3)
* 사용자 갤러리 및 공유 기능
* 평면도 평점 시스템 (리더보드 개념)
* 가구 쇼핑몰 연동
* AI 기반 공간 최적화 제안

## 5. User experience
### 주요 사용자 여정
* 첫 방문 시 간단한 튜토리얼 제공
* 빈 캔버스에서 시작하거나 템플릿 선택
* 좌측 툴바에서 벽 도구 선택 → 클릭으로 벽 그리기
* 우측 라이브러리에서 가구 드래그하여 배치
* 상단 메뉴에서 저장/공유/다운로드

### 엣지 케이스 및 UI 노트
* 브라우저 새로고침 시 작업 내용 임시 저장
* 모바일에서는 제한적 기능 제공 (보기 전용)
* 인터넷 연결 끊김 시 로컬 저장 기능
* 접근성을 위한 키보드 단축키 지원

## 6. Narrative
토요일 아침, 김민수는 커피를 마시며 새 아파트 평면도를 바라봅니다. "거실에 소파를 어디 놓을까?" 고민하던 그는 웹 브라우저를 열고 평면도 툴에 접속합니다. 

몇 번의 클릭만으로 아파트 벽체를 그리고, 드래그 앤 드롭으로 소파와 TV를 배치해봅니다. "여기 놓으면 동선이 막히겠네?" 다시 소파를 옮기고, 식탁 위치도 조정합니다. 

30분 후, 완성된 평면도를 아내에게 카카오톡으로 공유합니다. "여보, 이렇게 배치하면 어떨까?" 아내도 회사에서 링크를 열어 실시간으로 의견을 나눕니다. 주말 가구 쇼핑 전, 이미 완벽한 계획이 세워졌습니다.

## 7. Success metrics
### 사용자 참여 지표
* 월간 활성 사용자 수 (MAU)
* 평균 세션 시간 (목표: 20분 이상)
* 프로젝트 완료율 (저장까지 완료한 비율)
* 재방문률 (7일 내 재사용률)

### 비즈니스 지표
* 무료 회원에서 유료 전환율
* 평면도 공유 횟수
* 사용자 추천 점수 (NPS)
* 고객 지원 문의 감소율

## 8. Milestones & sequencing
### Phase 1 (3개월): MVP 출시
* 기본 평면도 그리기 기능
* 핵심 가구 라이브러리 (20개)
* 다운로드 기능
* 사용자 회원가입 및 프로젝트 저장

### Phase 2 (6개월): 기능 확장
* 협업 및 공유 기능
* 모바일 반응형 최적화
* 템플릿 라이브러리 구축
* 유료 플랜 도입

### Phase 3 (12개월): 플랫폼화
* 커뮤니티 갤러리 기능
* 파트너사 가구 연동
* AI 추천 시스템
* 리더보드 및 게임화 요소
